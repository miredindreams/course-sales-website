Частые вопросы о карьере

Ответы на популярные вопросы кандидатов

Какие технологии используются в компании?

Мы используем современный стек: React/TypeScript для фронтенда, Node.js/Python/Go для бэкенда, PostgreSQL/MongoDB для баз данных, Docker/Kubernetes для инфраструктуры. Каждая команда может выбирать оптимальные технологии для своих задач.

Есть ли возможность удаленной работы?

Да, мы remote-first компания. 85% сотрудников работают удаленно из разных стран. Для всех позиций доступна удаленная работа, кроме редких случаев, когда требуется присутствие в офисе.

Как проходит адаптация новых сотрудников?

У нас есть система адаптации: персональный buddy из команды, план на первые 90 дней, регулярные check-in встречи с менеджером. Первый месяц фокусируемся на знакомстве с командой и процессами.

Какие есть возможности для роста?

Ежегодные performance review, индивидуальный план развития, бюджет на обучение (конференции, курсы, сертификации), внутренние митапы и возможность менять команды/проекты внутри компании.
